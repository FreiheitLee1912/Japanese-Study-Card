# Japanese Study Card Generator

A simple web-based tool to create beautiful, printable Japanese study cards (cheat sheets) with Furigana support.

## How to Use

1.  **Upload to GitHub**: Upload this entire folder to a GitHub repository.
2.  **Enable GitHub Pages**: Go to Settings > Pages and set the source to the `main` branch (and root folder).
3.  **Open the App**: Visit your GitHub Pages URL (e.g., `https://yourname.github.io/repo-name/`).
4.  **Create Cards**:
    *   Fill in the form with your study content.
    *   Use `<ruby>漢<rt>かん</rt>字<rt>じ</rt></ruby>` for Furigana.
    *   Click "Generate Card" to see the result.
    *   Print the result to PDF or paper (A4 size).

## Files

*   `index.html`: The Admin Interface (Entry point).
*   `display.html`: The Card Display template.
*   `script.js`: Logic for saving data and opening the display page.
*   `style.css`: Styling for the card.
